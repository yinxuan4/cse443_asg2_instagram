# Instant Messaging (mobile/text chat) - Instagram

🛡️ Requirement 1: Real-Time Client-Side Content Moderation Filter
A high-performance, regex-driven moderation engine that intercepts and evaluates outgoing messages locally. By checking for scams, profanity, and spam links before network transmission, it ensures platform safety with zero server latency and provides instant, specific UI feedback to the user.

🎨 Requirement 2: Real-Time Collaborative Scribble Canvas
An interactive, shared drawing surface that allows users to sketch together simultaneously. By synchronizing pen strokes across different clients in real-time, it enables seamless, live visual communication and brainstorming directly within the chat environment.
